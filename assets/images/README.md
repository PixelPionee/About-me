## Especificações Técnicas

### Formato
- JPG para fotos/screenshots
- PNG para gráficos com transparência
- WebP para melhor performance (com fallback JPG)

### Otimização
- Qualidade: 80-85% para JPG
- Usar ferramentas de compressão:
  - [TinyPNG](https://tinypng.com/)
  - [Squoosh](https://squoosh.app/)
  - [ImageOptim](https://imageoptim.com/)

### Dimensões Recomendadas
- **Thumbnails:** 400x300px (4:3)
- **Hero Images:** 1200x600px (2:1)
- Manter aspect ratio consistente

## Placeholder Temporário

Enquanto as imagens reais não estão disponíveis, você pode usar:
- [Unsplash](https://unsplash.com/) - Fotos gratuitas de alta qualidade
- [Placeholder.com](https://placeholder.com/) - Geradores de placeholder
- Screenshots dos projetos reais quando disponíveis

## Exemplo de Uso

```html
<img src="/assets/images/ghost-hero.jpg" alt="Ghost of Tsushima Interactive Experience" loading="lazy">
```
